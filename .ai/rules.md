# AI Rules

## Stack

- Next.js App Router
- React 19
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- NestJS
- Prisma
- PostgreSQL
- TanStack Query
- Zustand
- React Hook Form
- Zod

## Architecture

- Feature Based
- Modular Monolith
- DTO First
- Service Layer
- Repository Pattern (when needed)

## Rules

- Never use any.
- Prefer composition over inheritance.
- Always create reusable components.
- Business logic never lives inside controllers.
- Validation must use Zod (frontend) and DTO (backend).
- Write readable code over clever code.
